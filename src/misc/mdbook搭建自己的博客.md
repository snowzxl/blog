# 使用 Mdbook 搭建自己的博客

## Mdbook 安装

我现在非常喜欢使用[Scoop](https://scoop.sh/)来安装和管理软件,它非常方便.

```sh
scoop install rust
scoop install mdbook
```

## 初始化你的目录

mdbook 支持`init`命令
`init`命令使用如下
```sh
mdbook init
```

第一次使用`init`

```sh
book-test/
├── book
└── src
    ├── chapter_1.md
    └── SUMMARY.md



chcp
├── book
└── src
    ├── chapter_1.md
    └── SUMMARY.md
    
```
